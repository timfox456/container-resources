# Openshift Labs 1.1, 1.2, 1.3
===============


### Prerequisites before starting:
 * A computer system with internet connectivity and web browser installed.
 * O/S: Windows, Mac, or Linux
 * A Github account.
 
 
 ### Labs will walk through these requirements:
 * Download and install Git software as described in Step 1 of Openshift Lab 1.1.
 * Sign up for Openshift Online in Step 2 of Openshift Lab 1.1.
 * Installing Openshift Command Line Interface in Step 1 of Openshift Lab 1.3.
